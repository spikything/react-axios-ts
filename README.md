# react-axios-ts

A demonstration of using Axios with 3 flavours of React:

- Class Components (legacy version)
- Functional Components (Hooks)
- Functional Components in Typescript

### Setup

Fork/clone the repo, then

```
npm i
```

### Install Axios

```
npm i axios
```

### Running

```
npm run dev
```

### Building

```
npm run build
```
